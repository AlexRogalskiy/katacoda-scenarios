# Install DVC

`pip3 install --ignore-installed PyYAML`{{execute}}

`pip3 install dvc`{{execute}}

`dvc --help`{{execute}}

`dvc version`{{execute}}

**Optional:** Install bash completion for `dvc`:

```
dvc completion -s bash | sudo tee /etc/bash_completion.d/dvc
```{{execute}}

`source /etc/bash_completion`{{execute}}
